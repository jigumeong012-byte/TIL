# java 강의 51~

## 1. 클래스 & 객체
- 클래스: 데이터와 기능을 묶은 설계도
- 객체: 클래스를 기반으로 생성된 실제 데이터
- `new` 키워드로 객체 생성
- 객체마다 서로 다른 값 저장 가능

---

## 2. 메서드
- 기능을 수행하는 코드 묶음
- 목적: 코드 재사용, 중복 제거
- 구조: 입력(매개변수) → 처리 → 반환(return)

---

## 3. 기본형 vs 참조형
- 기본형: 값을 직접 저장 (int, double 등)
- 참조형: 객체의 주소 저장 (String, 배열, 객체)

### 차이점
- 기본형: 서로 영향 없음
- 참조형: 같은 객체를 공유하면 값 변경 시 영향 있음

---

## 4. 생성자
- 객체 생성 시 자동 호출되는 메서드
- 초기값 설정에 사용
- 클래스 이름과 동일
- 반환 타입 없음

---

## 5. this
- 현재 객체 자신을 가리키는 키워드
- 변수 이름이 겹칠 때 구분용

---

## 6. 접근 제어자
- 데이터 접근 범위 설정

| 키워드   | 범위 |
|----------|------|
| public   | 어디서든 접근 가능 |
| private  | 클래스 내부에서만 가능 |
| default  | 같은 패키지에서만 가능 |

---

## 7. 캡슐화
- 데이터를 보호하기 위한 개념
- 변수는 `private`으로 숨김
- `getter/setter`로 접근

---

## 8. static
- 클래스에 소속된 변수/메서드
- 객체 없이 사용 가능
- 모든 객체가 공유

### 사용 목적
- 공용 데이터 관리
- 공용 기능 제공

---

## 9. final
- 값 변경 불가 (상수)
- 한 번 설정하면 수정할 수 없음
