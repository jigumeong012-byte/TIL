## back-end (백엔드)
백엔드란 웹사이트나 앱이 보이지 않는 뒤에서 작동하도록 만드는 서버 프로그램 부분을 말합니다.
또한 백엔드는 웹 서비스가 정상적으로 작동하도록 데이터 관리, 서버 로직 처리, 
보안 관리, 그리고 프론트엔드와의 데이터 통신(API) 등을 담당합니다.
이러한 기능을 통해 웹사이트나 앱이 안정적으로 동작할 수 있습니다.

### 백엔드의 주역할
*사용자 요청 처리
    * 저장 및 관리
         *로그인 / 회원가입 같은 인증 처리
               *서버에서 필요한 계산 및 로직 처리
                  *프론트엔드에 데이터(API) 제공

### 사용되는 기술
*Java
  *Node.js
    *Python
      *PHP
        *C#

### 프론트엔드 (front-end) vs 백엔드 (back-end)
프론트엔드란 사용자가 직접 보는 화면을 만드는 것을 말한다.
백엔드란 보이지 않는 서버 쪽 개발이다.

#### 프론트엔드가 주로 사용하는 기술
*HTML
  *CSS
    *JavaScript

#### 백엔드가 주로 사용하는 기술
*Java
    *Node.js
        *python
            *PHP
                *C#
